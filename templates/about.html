About the Syllabus Accessibility Checker

What is this tool?

The Syllabus Accessibility Checker is a comprehensive tool designed to help instructors create more accessible and inclusive course syllabi. It performs 40+ algorithmic checks to identify accessibility barriers that might prevent students from accessing course information.

Why does accessibility matter?

An accessible syllabus ensures that all students, including those who use assistive technologies like screen readers, can access critical course information. Accessibility benefits everyone:

  • Students with visual impairments using screen readers
  • Students with dyslexia or cognitive disabilities
  • Students with color vision deficiencies
  • International students learning in a second language
  • Students using mobile devices or slow internet connections

What does it check?

The tool performs 40+ checks across multiple categories:

Tables (10 checks)
  • Layout tables vs. data tables
  • Missing header rows
  • Table captions and descriptions
  • Merged cells and reading order
  • Color-coded meaning
Typography (8 checks)
  • Font sizes and readability
  • Decorative fonts
  • Font consistency
  • Line spacing
  • ALL CAPS blocks
Color & Contrast (3 checks)
  • WCAG contrast requirements
  • Color as sole indicator
  • Text over backgrounds
Links & Navigation (7 checks)
  • Descriptive link text
  • Link styling
  • Long URLs
  • Table of contents
  • Bookmarks
Images (3 checks)
  • Missing alt text
  • Decorative marking
  • Image presence verification
Lists (3 checks)
  • Manual vs. built-in formatting
  • List hierarchy
  • Layout misuse

How to use the tool

  1. Upload your syllabus (.docx format)
  2. Review the detailed accessibility report
  3. Download the marked document with issues highlighted
  4. Fix the issues in your original document
  5. Run the check again to verify improvements

Standards and Guidelines

This tool is based on:

  • WCAG 2.1/2.2 - Web Content Accessibility Guidelines
  • PDF/UA - Universal Accessibility standards
  • Section 508 - U.S. accessibility requirements
  • Best practices for educational materials
Note

This tool performs algorithmic checks and may not catch all accessibility issues. Manual review and user testing are still important for ensuring full accessibility.
